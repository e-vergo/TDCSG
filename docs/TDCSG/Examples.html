    Examples of Piecewise Isometries #

    This file provides concrete examples of piecewise isometries and counterexamples of functions that fail to be piecewise isometries.

    Main examples #

    Proven piecewise isometries: identity map, half-plane reflection, simple square billiard.

    Proven non-isometries: constant functions, doubling map, Baker's map.
